### 1.1 Существующие компоненты
- **KworkClient** (`/workspace/kwork/client.py`) - клиент для работы с Kwork API
- **Config** (`/workspace/config.py`) - настройки подключения к Kwork

- **Логика бота** (`/workspace/bot/`) - взаимодействие с пользователем
